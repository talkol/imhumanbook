# Why I’m Writing This Book

This book is my attempt to develop an elegant mental model that can explain human life. I accept that many different mental models can coexist and all can be subjectively true.

It’s important to emphasize that this mental model only applies to my own subjective view of reality. I share it with you with the hope that your feedback will help me improve it.

The most elegant mental model logically has the following properties:

- The simplest and most minimal set of assumptions.
- Fits as many observations about life as possible.
- All deductions are based on pure logic and will easily convince any logical reviewer that accepts the same set of assumptions.

This book is forever work in progress because I can never assume that I have found the most elegant model. I assume that the mental model can forever be improved.

If any of the readers succeeds in developing a more elegant model that requires a smaller set of assumptions, I ask that you please publish your work. I for one will be very interested to study it.

I decided to publish this book on Github as it’s being written. This will allow myself and any interested fellow reviewers to track the progress and see all revisions.
